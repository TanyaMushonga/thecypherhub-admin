# Contributing to The Cypher Hub Admin

Thank you for considering contributing to The Cypher Hub Admin! We welcome contributions from everyone.

## Getting Started

### 1. Fork the Repository
Click the "Fork" button at the top right of this repository to create your own copy of the project.

### 2. Clone Your Fork
Clone your forked repository to your local machine:
```bash
git clone https://github.com/YOUR_USERNAME/thecypherhub-admin.git
cd thecypherhub-admin
```

### 3. Create a Branch
Create a new branch for your feature or bug fix:
```bash
git checkout -b feature/my-new-feature
# or
git checkout -b fix/my-bug-fix
```

### 4. Install Dependencies
Make sure you have Node.js installed, then run:
```bash
npm install
```

### 5. Set Up Environment Variables
Copy the example environment variables (or see README) and configure your `.env` file with your local database and API keys.

### 6. Run Development Server
```bash
npm run dev
```

## Making Changes

- **Code Style:** We use Prettier and ESLint. Please ensure your code is formatted before committing.
- **Commits:** Write clear, descriptive commit messages.
  - GOOD: `feat: add user profile page`
  - BAD: `updated stuff`

## Submitting a Pull Request

1. Push your changes to your fork:
